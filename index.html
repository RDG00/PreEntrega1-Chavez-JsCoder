Simulador Interactivo

Simulador de compra de entradas


Tengan buen día y sean bienvenidos a mi primera entrega del Curso JS, para este proyecto quise realizar un simulador de compra para un recital con capacidad máxima de 150 personas, con requerimiento que el comprador sea mayor de edad, si es menor, la compra no será posible.


A medida que se van llenando las cuotas de las 150 entradas vendidas, el código irá sumando hasta llegar al tope, y para las próximas compras ya no serán posibles
